# UserAuth-Django
Basic user authentication using django

## Requirements

- Python 3.6 and above
- pip3 for Python 3.6 and above


---


## Setup

1. Run the Django server

```
pip install -r requirements.txt
python3 manage.py runserver
```

_Runs default on port 8000
